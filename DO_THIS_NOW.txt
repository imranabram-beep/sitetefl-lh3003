✅ STEP 2: Create Supabase Client (2 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Create a new file: lib/supabase.ts

Copy and paste this code:

  import { createClient } from "@supabase/supabase-js";

  const supabaseUrl = process.env.NEXT_PUBLIC_SUPABASE_URL || "";
  const supabaseKey = process.env.SUPABASE_SERVICE_ROLE_KEY || "";

  export const supabase = createClient(supabaseUrl, supabaseKey);

Save the file.

   - From Email: noreply@siteTeFL.com
   - From Name: siteTeFL

   - Name: siteTeFL-admin
   - Permissions: Full Access
10. Copy the API key immediately (you won't see it again!)

12. Add Endpoint: http://localhost:3003/api/unsubscribe (for local testing)
    (Use your domain URL for production)
13. Enable: Opened, Clicked, Bounced, Unsubscribe, Dropped

✅ STEP 5: Update .env.local (3 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Add these lines to your .env.local file:

  # Supabase
  NEXT_PUBLIC_SUPABASE_URL=https://your-project-id.supabase.co
  SUPABASE_SERVICE_ROLE_KEY=eyJhbGc...

  # SendGrid
  SENDGRID_API_KEY=SG.xxxxxxxxxxxxxxxx_xxxxxxxxxxxxxxxx
  SENDGRID_FROM_EMAIL=noreply@siteTeFL.com

  # App URL
  NEXT_PUBLIC_APP_URL=http://localhost:3003

Replace:
  - your-project-id with actual ID from Supabase
  - eyJhbGc... with your Service Role Key from Supabase
  - SG.xxx... with your SendGrid API key
  - noreply@siteTeFL.com with your verified sender email

Save the file.

⚠️ IMPORTANT: Restart your dev server!

  npm run dev
