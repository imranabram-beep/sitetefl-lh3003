🔴 API #1: USAJobs (USA) - REGISTER NOW

   Step 1: Go to https://developer.usajobs.gov/
   Step 2: Sign up → Verify email → Login
   Step 3: Go to API Keys section
   Step 4: Create new API key → Copy it

   Step 5: Update your .env.local file:
           Open: /sessions/blissful-kind-ritchie/mnt/siteTEFL LH3003/.env.local
           Find: USAJOBS_API_KEY=your_api_key_here
           Replace with: USAJOBS_API_KEY=<paste-your-key-here>
           Also set: USAJOBS_USER_AGENT=your_email@example.com
           Save file

   Time: 5 minutes
   Status: __________


🔴 API #2: Bundesagentur (Germany) - REGISTER NOW

   Step 1: Go to https://github.com/bundesAPI/jobsuche-api
   Step 2: Find and click the registration link
   Step 3: Complete registration
   Step 4: Receive API key by email (within minutes)

   Step 5: Update your .env.local file:
           Find: ARBEITSAGENTUR_API_KEY=your_api_key_here
           Replace with: ARBEITSAGENTUR_API_KEY=<paste-your-key-here>
           Save file

   Time: 5 minutes
   Status: __________


🔴 API #3: France Travail (France) - REGISTER NOW

   Step 1: Go to https://francetravail.io/
   Step 2: Click "S'inscrire" (Sign up - top right)
   Step 3: Create account → Verify email → Login
   Step 4: Go to API/Application section
   Step 5: Create new application
   Step 6: Copy client_id and client_secret

   Step 7: Update your .env.local file:
           Find: FRANCE_TRAVAIL_CLIENT_ID=your_client_id_here
           Replace with: FRANCE_TRAVAIL_CLIENT_ID=<paste-your-id-here>

           Find: FRANCE_TRAVAIL_CLIENT_SECRET=your_client_secret_here
           Replace with: FRANCE_TRAVAIL_CLIENT_SECRET=<paste-your-secret-here>
           Save file

   Time: 10 minutes
   Status: __________
